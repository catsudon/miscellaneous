# git cheat sheet

add new remote
```
git remote add <origin> <url>
```

pull changes done on remote (but not on local)
```
git pull <origin> <branch>
```

view changes not staged for commit 
** don't for get to change the branch to main **
```
git status
```

view changes
```
git diff <file/path>
```

merge with current branch
```
git merge <branch>
```
